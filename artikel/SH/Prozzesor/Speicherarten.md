# Datenspeicher in der Computertechnik

Speicher werden nach ihrer Flüchtigkeit (bleiben Daten ohne Strom erhalten?) und ihrer Zugriffsart unterschieden.

## RAM (Random Access Memory)
Flüchtiger Speicher, der als Arbeitsspeicher dient.
- **SRAM (Static RAM):** Schnell, teuer, braucht viel Platz (Transistoren). Einsatz als Cache.
- **DRAM (Dynamic RAM):** Günstig, hohe Dichte (Kondensatoren), muss ständig aufgefrischt werden (Refresh).

## ROM (Read Only Memory)
Nicht-flüchtiger Speicher.
- **EEPROM:** Elektrisch lösch- und beschreibbar, aber langsam beim Schreiben.
- **Flash-Speicher:** Weiterentwicklung des EEPROM (z. B. SSDs, USB-Sticks). Daten werden in Blöcken gelöscht.

## Memory Map
Die Memory Map legt fest, unter welcher Adresse die CPU den RAM, das ROM oder die Ein-/Ausgabegeräte (Peripherie) findet.
